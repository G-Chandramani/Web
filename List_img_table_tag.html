Image Tag

Image tag Used for adding images to the webpage. it having src attribute for giving the source location for image wee can able to give online image link + local path for the image. It also have width and height attribute for adjusting width and height for image. The alt(alternative text) attribute is used for the showing alternative text if our image is not visible on the webpage
